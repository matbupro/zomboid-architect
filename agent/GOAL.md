# Objectif Principal — Zomboid_Architect

Projet : **Project Zomboid Knowledge Engine**
Créateur : **ElChibros**

Construire un moteur de connaissance local, déterministe et sans hallucination sur Project Zomboid, exposé à un agent IA via un serveur MCP.

Il sert deux buts simultanés :
1. **Stratégie de survie** — conseils précis (calories, panique, moodles, bruit, agriculture…)
2. **Développement de mods** — documentation Lua/Java, génération de templates de mod

Cycles de version : alpha → beta → release candidate → release
